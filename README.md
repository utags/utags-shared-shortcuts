# utags-shared-shortcuts

Shared shortcuts configuration for [UTags Shortcuts](https://github.com/utags/userscripts/tree/main/utags-shortcuts).

[简体中文 (zh-CN)](./zh-CN/README.md)

## Introduction

This repo is for co-building and sharing UTags Shortcuts.
Everyone is welcome to submit PRs to modify or add new groups and shortcuts.

## Import

1. Copy the **Raw** link of the JSON file you want to import from the [list](./zh-CN/README.md).
2. Open UTags Shortcuts settings and go to the "**Data Management**" tab.
3. In the "**Data Import**" section, click "**Import from URL**" and paste the link.
    * You can also click "**Import from JSON file**" or "**Import from pasted text**" to import the downloaded JSON file or content.
