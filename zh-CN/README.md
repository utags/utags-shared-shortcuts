# Shared Shortcuts (zh-CN)

这个 repo 是共建并分享 [UTags Shortcuts](https://github.com/utags/userscripts/tree/main/utags-shortcuts) 的配置。
大家可以提交 PR ，修改或新添加分组与 shortcuts。

## 导入方法

1. **复制链接**：在下方表格中找到需要的分组，右键点击“下载链接”并复制链接地址（Raw URL）。
2. **打开设置**：打开 UTags Shortcuts 的设置面板，进入“**数据管理**”标签页。
3. **导入配置**：在“**数据导入**”区域，点击“**从 URL 导入**”按钮，粘贴链接地址即可导入。
   - 也可以点击“**从 JSON 文件导入**”或“**粘贴文本导入**”按钮来导入下载的 JSON 文件或内容。
